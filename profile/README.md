## Overview

ToothFairy Advanced is a one-click Bluetooth connection manager from C-Command that provides instant connection and disconnection of Bluetooth devices — AirPods, headphones, keyboards, mice, and other peripherals — from a menu bar icon or keyboard shortcut, eliminating the multi-step process of opening Bluetooth preferences to manage connections. It solves the everyday frustration of connecting AirPods and other Bluetooth devices on macOS, where the built-in interface requires navigating through System Settings to establish a connection.

One-click connection assigns a menu bar icon and optional keyboard shortcut to each configured Bluetooth device, enabling instant connection with a single click or keystroke. The connection is immediate — no settings windows, no menus to navigate, no waiting for Bluetooth preference panels to load. Disconnection is equally instant with the same single action.

Menu bar status displays the connection state and battery level of configured devices directly in the menu bar, providing at-a-glance visibility into device status without opening any settings. Custom icons for each device make identification instant in multi-device configurations. The battery level display eliminates the surprise of dead headphones.

Advanced audio codec selection forces specific Bluetooth audio codecs — AAC, aptX, LDAC — when connecting audio devices, ensuring the highest quality audio codec is used rather than defaulting to a lower-quality option. This capability addresses the macOS behavior of sometimes connecting audio devices with suboptimal codec selection.

For AirPods users and anyone who connects and disconnects Bluetooth devices frequently and finds the built-in macOS Bluetooth interface too slow and cumbersome, ToothFairy Advanced is the connection tool that makes Bluetooth device management as fast as clicking an icon.

## Performance

ToothFairy balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
